3 casos de uso que transformam
o canal WhatsApp da CAIXA.

Cada par de telas confronta a experiência de hoje com a proposta Blip + gw.cloud. A Blip constrói as jornadas conversacionais com IA e acessa a base de dados da CAIXA; a gw.cloud autentica o usuário via biometria facial, garantindo segurança e prevenindo fraudes. Nenhum saldo ou transação é exibido sem autenticação prévia.

Plataforma Blip
Jornadas + IA + integrações
A plataforma Blip é responsável pela construção de todas as jornadas conversacionais com IA e todas as integrações com sistemas CAIXA (CRM, motor de crédito, Dataprev, CTPS Digital, gov.br). A orquestração é feita no Blip Studio.
Camada gw.cloud
Biometria facial + anti-fraude via API
A solução gw.cloud é acionada pelo Blip Studio via API para validar dados cadastrais e coletar biometria facial em tempo real. Autentica o usuário sem dependência do App CAIXA, SMS ou cartão físico. Acionada em operações sensíveis — consulta de saldo, contratação, transações.
